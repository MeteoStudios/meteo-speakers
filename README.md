## Install

1. Drop the resource into your `resources` folder.
2. `ensure meteo-speakers`
3. Add Items to Inventory - check `meteo-speakers/install/qbcore/qb-items.md` or `meteo-speakers/install/qbox/ox-items.md`
4. Add Item Images Copy all images from the `meteo-speakers/install/images` to your inventory images folder. 

## Requirements

- [Meteo Phone](https://docs.meteofivem.net/paid-scripts/meteo-phone)
- `ox_lib`
